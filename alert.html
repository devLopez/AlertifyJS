Alert Dialog

An alert dialog is often used if you want to make sure information comes through to the user. When an alert dialog pops up, the user will have to click "OK" to proceed.
AlertifyJS

Alert

OK

Default usage

You create an alert dialog by invoking alertify.alert(...)

Calling alertify.alert() without passing any parameter will return the alert dialog instance. You can use this syntax to modify dialog settings before showing it. Also you can get the dialog instance (singletons only) by invoking alertify.dialog('alert'); 


/*
 * @title {String or DOMElement} The dialog title.
 * @message {String or DOMElement} The dialog contents.
 * @onok {Function} Invoked when the user clicks OK button or closes the dialog.
 *
 * alertify.alert(title, message, onok);
 * 
 * overloads:
 *
 * alertify.alert(message);
 *
 * alertify.alert(title, message);
 *
 * alertify.alert(message, onok);
 *
 */
alertify.alert('This is an alert message!');
Run

API reference

message { String or DOMElement }

Description: Sets the dialog contents

Default: undefined

 alertify.alert().set('message', 'This is a new message!').show();
Run

label { String }

Description: Sets the OK button label

Default: alertify.defaults.glossary.ok

 alertify.alert('label changed!').set('label', 'Alright!');
Run

onok { Function }

Description: Gets or sets a callback function to be invoked when the user clicks 'Ok' button or closes the dialog.

Default: undefined

 alertify.alert('a callback will be invoked on ok.').set('onok', function(){ alertify.success('Ok');} );
Run



You can use the .setting() method to set multiple values at once, Also works as a getter. Or use .get() and .set() aliases. 


//get the closable setting value.
var closable = alertify.alert().setting('closable');
//grab the dialog instance and set multiple settings at once.
alertify.alert()
  .setting({
    'labels':{ok:'Yup', cancel:'Nop'},
    'message': 'This dialog is : ' + (closable ? ' ' : ' not ') + 'closable.' ,
    'onok': function(){ alertify.success('Great');}
  }).show();
Run

API reference (Common)

This refers to settings/methods that are common to all dialogs.

modal {Boolean}

Description: Sets the dialog display mode

Default: true

 alertify.alert('Modal: false').set('modal', false);
Run

basic {Boolean}

Description: Sets the dialog basic view mode, which hides both header/footer reserving their space as not to affect other options such as resize, move, maximize...etc.

Default: false

 alertify.alert('Basic: true').set('basic', true);
Run

frameless {Boolean}

Description: Sets the dialog frameless view mode, which hides both header/footer without reserving their space, the content author must be aware that dialog commands might overlap with contents.

Default: false

 alertify.alert('Frameless: true').set('frameless', true);
Run

movable {Boolean}

Description: Gets or sets a value indicating whether the dialog is movable.

Default: true

 alertify.alert('Movable: false').set('movable', false);
Run

resizable {Boolean}

Description: Gets or sets a value indicating whether the dialog is resizable.

Default: false

 alertify.alert('Resizable: true').set('resizable', true);
Run

closable {Boolean}

Description: Gets or sets a value indicating whether the Close button is displayed in the header of the dialog. (clicking the dimmer will also trigger a close command)

Default: true

 alertify.alert('Closable: false').set('closable', false);
Run

maximizable {Boolean}

Description: Gets or sets a value indicating whether the Maximize button is displayed in the header of the dialog.

Default: true

 alertify.alert('Maximizable: false').set('maximizable', false);
Run

startMaximized {Boolean}

Description: Gets or sets a value indicating whether the dialog should start in a maximized state or not.

Default: false

 alertify.alert().set({'startMaximized':true, 'message':'Start Maximized: true'}).show();
Run

pinnable {Boolean}

Description: Gets or sets a value indicating whether the Pin button is displayed in the header of the dialog (modeless only).

Default: true

 alertify.alert('Modeless: false, Pinnable: false. Try scrolling the page!').set({'pinnable': false, 'modal':true});
Run

transition {String}

Description: Gets or sets the transition effect to be used when showing/hiding the dialog

Default: alertify.defaults.transition:pulse

 alertify.dialog('alert').set({transition:'pulse',message: 'Transition effect: pulse'}).show();

onshow {Function}

Description: Gets or sets a function to invoke when the dialog is shown

Default: undefined

 alertify.alert().set({onshow:function(){ alertify.message('alert was shown.')}});
 alertify.alert('Demo');
Run

onclose {Function}

Description: Gets or sets a function to invoke when the dialog is closed

Default: undefined

 alertify.alert('Demo').set({onshow:null, onclose:function(){ alertify.message('alert was closed.')}});
Run

onfocus {Function}

Description: Gets or sets a function to invoke when the dialog is focused for the first time.

Default: undefined

 alertify.alert('Demo').set({onfocus:function(){ alertify.message('alert onfocus callback.')},onshow:null, onclose:null});
Run

overflow {Boolean}

Description: Gets or sets a value indicating whether the content overflow is managed by the dialog or not.

Default: true

 alertify.alert().set('overflow',false);

padding {Boolean}

Description: Gets or sets a value indicating whether the content padding is managed by the dialog or not.

Default: true

 alertify.alert().set('padding',false);

maximize ( ) chainable

Description: Maximizes the dialog.

Parameters: none

 alertify.alert('This is a maximized dialog').maximize();
Run

restore ( ) chainable

Description: Restores the maximized dialog.

Parameters: none

 var div = document.createElement('div');
 div.innerHTML = 'This will be resored in 3 seconds';
 //show maximized dialog
 var alert = alertify.alert(div).maximize();
 //restore the dialog after 3 seconds
 var i = 3;
 var interval = setInterval(function(){
   if(--i == 0){
     clearInterval(interval);
     //restore the dialog
     alert.restore().set('message','Dialog restored.');
   }else{
     div.innerHTML = 'This will be resored in ' + i + ' seconds.';
   }
 }, 1000);
Run

pin ( ) chainable

Description: Pins a modeless dialog to the screen, preventing it from moving with body scroll.

Parameters: none

 alertify.alert('This is a modeless dialog, pinned to the screen.').set('modal', false).pin();
Run

unpin ( ) chainable

Description: Un-pins a modeless dialog from the screen, allowing it to move with body scroll.

Parameters: none

 alertify.alert('This is a modeless dialog, not pinned to the screen.').set('modal', false).unpin();
Run

isOpen ( ) {Boolean}

Description: Gets a value that indicates whether the dialog is open.

Parameters: none

 var isOpen = alertify.alert().isOpen();

isModal ( ) {Boolean}

Description: Gets a value that indicates whether the dialog is modal.

Parameters: none

 var isModal = alertify.alert().isModal();

isMaximized ( ) {Boolean}

Description: Gets a value that indicates whether the dialog is maximized.

Parameters: none

 var isModal = alertify.alert().isMaximized();

isPinned ( ) {Boolean}

Description: Gets a value that indicates whether the dialog is pinned.

Parameters: none

 var isModal = alertify.alert().isPinned();

closeOthers ( ) chainable

Use alertify.closeAll() to close all open dialogs.

Description: Closes all open dialogs except this one.

Parameters: none

 alertify.alert().closeOthers();

moveTo ( x, y ) chainable

Description: Move the dialog to a specific X,Y coordinates.

Parameters:

@x {Number} X coordinate in pixels.

@y {Number} Y coordinate in pixels.

 alertify.alert('Moved to 0,0').moveTo(0,0);
Run

resizeTo ( width, height ) chainable

The dialog must be resizable and it can be resized to a minimum width equal to the initial display width and a minimum height equal to the sum of header and footer heights.

Description: Resize the dialog to a specific width/height (the dialog must be 'resizable').

Parameters:

@width {Number} The new dialog width in pixels.

@height {Number} The new dialog height in pixels.

 alertify.alert('Resized to 600,450').set('resizable',true).resizeTo(600,450);
Run
